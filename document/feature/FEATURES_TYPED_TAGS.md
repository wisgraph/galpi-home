# 🏷️ Typed Tags (타입드 태그) 가이드

Hooklink의 **Typed Tag** 시스템은 단순한 키워드 분류를 넘어, 마치 프로그래밍 언어의 라이브러리처럼 데이터를 체계적으로 구조화할 수 있도록 돕는 혁신적인 태그 시스템입니다.

---

## 🚀 1. Typed Tag란 무엇인가요?

일반적인 태그가 단순히 단어의 나열이라면, Typed Tag는 `타입(Type)`과 `값(Value)`의 조합으로 이루어집니다. `[스크린샷: 검색창에 #project:hooklink와 같은 타입드 태그가 입력된 모습]`

*   **구조:** `#타입:값` (예: `#project:hooklink`, `#status:doing`)
*   **특징:** 타입별로 고유한 색상이 부여되어 검색과 관리가 시각적으로 명확해집니다.

---

## 💡 2. 지능형 자동완성 (IDE-Style Autocomplete)

Hooklink는 마치 코드 에디터처럼 사용자의 입력 문맥을 이해합니다. `[스크린샷: 검색창에서 #을 입력했을 때 타입 추천 리스트가 나타나는 모습]`

1.  **타입 추천:** `#`만 입력하면 `project:`, `status:`, `priority:` 등 미리 정의된 프리셋과 사용자가 직접 만든 타입들이 나타납니다.
2.  **값 추천:** `#project:` 처럼 콜론까지 입력하면, 해당 프로젝트 타입으로 등록된 기존 값들만 필터링하여 보여줍니다. `[스크린샷: #project: 입력 후 등록된 프로젝트 목록이 제안되는 모습]`
3.  **스마트 타이핑:** 타입을 선택할 때는 띄어쓰기를 하지 않고 바로 값을 입력하도록 가이드하며, 태그 입력이 완료되면 자동으로 한 칸 띄워주어 연속적인 입력이 가능합니다.

---

## 🎨 3. 나만의 태그 스타일 커스터마이징

Typed Tag의 심미성은 사용자가 직접 제어할 수 있습니다. `[스크린샷: 설정 화면의 Tag Styles 섹션에서 색상을 변경하는 모습]`

*   **자동 탐색:** 새로운 타입(`예: #platform:ios`)을 입력하면 시스템이 이를 자동으로 인식하여 스타일 설정 목록에 추가합니다.
*   **커스텀 컬러:** 설정 화면의 **Tag Styles** 섹션에서 각 타입의 인디케이터 색상을 원하는 대로 변경할 수 있습니다.
*   **시각적 직관성:** 검색 결과에서도 각 태그의 타입은 라벨 형태로 표시되어, 필터링된 데이터의 성격을 한눈에 파악할 수 있습니다.

---

## 🛠️ 4. 주요 프리셋 활용 사례

Hooklink에서 기본으로 제공하는 프리셋을 활용해 데이터의 위계를 세워보세요.

*   **`#project:`** 진행 중인 업무나 업무 단위별 분류
*   **`#status:`** 진행 중, 완료, 보류 등 진행 상태 관리
*   **`#priority:`** 오늘 꼭 해야 할 일의 우선순위 설정
*   **`#date:`** 마감일이나 특정 날짜 기록

---

## 🌟 Typed Tag가 선사하는 변화

단순히 "찾는 것"이 아니라, 데이터가 어떤 맥락에 있는지 "읽는 것"이 가능해집니다. Hooklink의 Typed Tag와 함께 더 깊이 있는 데이터 경험을 시작해 보세요.
