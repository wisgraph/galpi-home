# Hooklink: Intelligent Tag Autocomplete
> **태그를 입력하는 순간, 당신의 데이터 지도가 완성됩니다.**

Hooklink의 **지능형 태그 자동완성(Tag Autocomplete)**은 사용자가 데이터를 분류하고 찾는 과정에서 단 1초도 낭비되지 않도록 설계되었습니다. 단순한 텍스트 제안을 넘어, 작업의 워크플로우를 이해하고 맥락을 연결하는 핵심 기술입니다.

---

## 1. 엠비언트 자동완성 (Ambient Autocomplete)

어떤 입력창에서든 `#` 문자를 입력하는 순간, Hooklink의 자동완성 엔진이 즉시 가동됩니다. 수천 개의 태그 중 현재 입력한 파편과 가장 일치하는 태그를 실시간으로 제안합니다.

### 🔍 메인 검색창 (Search Bar)
- **목적:** 전역 필터링 및 환경 구성
- **특징:** 특정 프로젝트나 카테고리로 데이터셋을 순식간에 좁히고 싶을 때 사용합니다. 태그를 선택하는 즉시 검색 쿼리가 완성되며 리스트가 반응합니다.

[스크린샷: 검색창에서 #을 입력했을 때 하단에 세련되게 나타나는 태그 제안 드롭다운 모달]

### ✏️ 이름 및 태그 수정 (In-place Renaming)
- **목적:** 데이터 수집 즉시 메타데이터 최적화
- **특징:** 훅의 이름을 수정하거나 새로운 태그를 추가할 때, 검색창과 동일한 자동완성 인터페이스가 제공됩니다. 별도의 대화상자 없이 리스트 내에서 바로 태그를 정제할 수 있습니다.

[영상: 훅의 제목을 수정하다가 #을 입력하여 기존 태그를 빠르게 선택하고 적용하는 편집 과정]

---

## 2. 파워 유저를 위한 '컨텍스트 승격'

Hooklink 자동완성만의 독보적인 기능은 **자동완성 단계에서 바로 컨텍스트 태그를 등록**할 수 있다는 점입니다.

- **워크플로우:** `#project_name`을 입력하고 태그가 추천된 상태에서 `Tab`이나 `Enter` 대신 **`Cmd + Enter`**를 누르세요.
- **결과:** 해당 태그가 검색어에 입력되는 대신, 즉시 **활성화된 컨텍스트(Active Context)**로 등록됩니다. 
- **효과:** 검색 모드에서 작업 모드로의 전환이 키보드 단축키 한 번으로 끝납니다.

[영상: 자동완성 목록에서 특정 태그를 Cmd+Enter로 선택하자, 검색창 우측 컨텍스트 영역에 해당 태그가 칩 형태로 고정되는 모습]

---

## 3. 기술적 특징 (Technical Highlights)

- **리얼타임 트리거 포지션:** 커서의 위치를 정확히 계산하여 텍스트 중간에서도 정확한 위치에 제안 목록을 띄웁니다.
- **퍼지 매칭 (Fuzzy Matching):** 오타가 있거나 태그의 일부만 입력해도 뒤에 숨겨진 의도를 파악하여 관련 태그를 우선순위대로 배치합니다.
- **키보드 퍼스트 디자인:** 마우스에 손을 올리지 않고도 방향키와 단축키(`Tab`, `Enter`, `Cmd+Enter`)만으로 모든 태그 관리 활동이 가능합니다.

---

## 4. 활용 팁

1. **태그 탐색:** 아무 글자도 입력하지 않고 `#`만 입력하면 가장 자주 사용하거나 최근에 사용한 태그 목록을 훑어볼 수 있습니다.
2. **다중 컨텍스트 등록:** `Cmd+Enter`를 연타하여 여러 개의 태그를 동시에 컨텍스트로 띄워두고 멀티 태스킹 환경을 구축하세요.

[스크린샷: 여러 개의 태그가 컨텍스트로 등록되어 복합적인 작업 환경이 구성된 화면]

---

**Hooklink의 태그 자동완성은 단순히 글자를 채워주는 기능이 아닙니다. 당신의 지식 파편들을 가장 빠른 경로로 연결하는 신경망입니다.**
