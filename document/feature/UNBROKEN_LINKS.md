# 끊어지지 않는 연결 (Unbroken Links): HookLink의 핵심 기술

누구나 한 번쯤 겪어봤을 겁니다.
파일 정리를 위해 폴더 이름을 바꾸거나, 다운로드 폴더의 파일을 문서 폴더로 옮겼을 뿐인데...
지난주에 저장해 둔 링크를 클릭하면 **"파일을 찾을 수 없습니다"**라는 에러 메시지가 뜨는 상황을요.

일반적인 링크는 **주소(Path)**를 기억합니다. 그래서 주소가 조금만 바뀌어도 길을 잃습니다.
하지만 **HookLink는 파일을 기억합니다.**

우리는 macOS의 강력한 파일 시스템 기술을 활용하여, 파일을 옮겨도, 이름을 바꿔도, 폴더 구조를 완전히 뜯어고쳐도 **절대 끊어지지 않는** 링크 기술을 구현했습니다.

---

## 1. 주소가 아닌 '파일 자체'를 추적합니다

기존 앱(Notion, 옵시디언 등)들은 링크를 단순한 텍스트 주소(예: `/Users/me/Documents/Report.pdf`)로 저장합니다. 이것이 "구식 방식"입니다.

HookLink는 **"이중 추적 시스템"**을 사용합니다.

### 디지털 지문 (Digital Fingerprint)
HookLink로 링크를 만드는 순간, 우리는 해당 파일만의 고유한 **디지털 지문**(`BookmarkData`)을 생성하여 저장합니다.

*   **위치가 아닌 정체성(Identity):** 이 지문은 파일 이름이나 위치가 아니라, 디스크 내의 고유 식별자(Inode)를 추적합니다.
*   **강력한 내구성:**
    *   파일 이름을 바꿔도? **연결됩니다.**
    *   다른 폴더로 이동해도? **연결됩니다.**
    *   상위 폴더 이름을 바꿔도? **여전히 연결됩니다.**

## 2. 자가 치유 (Self-Healing) 아키텍처

링크가 끊어진 것처럼 보이는 순간, HookLink의 진가가 발휘됩니다.
클릭 한 번에 일어나는 **"마법 같은 복구 과정"**은 다음과 같습니다.

### 자동 복구 프로세스
1.  **즉시 감지:** 사용자가 링크를 클릭하면, HookLink는 0.001초 만에 파일이 원래 위치에 있는지 확인합니다.
2.  **유령 추적 (Ghost Tracking):** 파일이 없다면, 저장해 둔 '디지털 지문'을 사용하여 macOS 파일 시스템에 묻습니다. *"이 파일, 어디로 이사 갔니?"*
3.  **위치 포착:** 운영체제(OS)는 즉시 파일의 새로운 주소를 알려줍니다. (예: `/Downloads` -> `/Projects/Archive`)
4.  **자가 치유 (Self-Healing):**
    *   파일을 즉시 엽니다. (사용자는 지연을 느끼지 못합니다.)
    *   **중요:** HookLink는 내부 데이터베이스(JSONL)의 주소를 **새로운 위치로 스스로 업데이트**합니다.
    *   이제 링크는 영구적으로 고쳐졌습니다.

## 3. 이것이 왜 중요한가요?

단순한 기능이 아닙니다. 이것은 당신의 지식 관리 방식을 근본적으로 바꿔줍니다.

*   **자유로운 정리 (Freedom to Organize):** *"링크 깨지면 어떡하지?"*라는 걱정 없이 언제든 폴더 구조를 개선하고 정리하세요.
*   **유지보수 제로 (Zero Maintenance):** 깨진 링크를 일일이 수정하느라 시간을 낭비할 필요가 없습니다. HookLink는 스스로를 관리합니다.
*   **네이티브 성능:** Finder가 사용하는 것과 동일한 애플 공식 기술을 사용하여, 배터리 소모 없이 빠르고 안정적으로 작동합니다.

## 요약

| 기능 | 일반적인 링크 (타사 앱) | HookLink |
| :--- | :--- | :--- |
| **추적 방식** | 텍스트 주소 (경로) | **디지털 지문 (BookmarkData)** |
| **파일 이동 시** | ❌ 링크 끊어짐 | ✅ **링크 자동 연결** |
| **이름 변경 시** | ❌ 링크 끊어짐 | ✅ **링크 자동 연결** |
| **관리 방식** | 사용자가 수동 수정 | **앱이 알아서 자가 치유** |

**HookLink는 당신의 자료가 어디에 있든, 언제나 그 자리에서 당신을 기다립니다.**
