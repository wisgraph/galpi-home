
### 1. 히로 섹션 (The Hero): "비교의 미학"

가장 먼저 보여줄 것은 말씀하신 **영상(Split Screen Video)** 입니다. 백마디 말보다 강력합니다.

- **레이아웃:** 좌우 분할 화면 (Split Screen)
- **좌측 화면 (The Old Way - 🐢):**
    - 마우스 커서가 바쁘게 움직임 → 크롬 북마크바 클릭 → '업무' 폴더 클릭 → 스크롤 주르륵 → '참고자료' 클릭 → (한숨).
    - 동시에 파인더 열기 → 문서 폴더 더블클릭 → 프로젝트 폴더 더블클릭 → 파일 찾기... (총 12초 소요)
    - **캡션:** _"익숙하지만 느린 방식 (12초)"_
- **우측 화면 (HookLink Way - ⚡️):**
    - `Opt+Space` (런처 뜸) → `ㄱㅎ` (기획) 입력 → `Enter` → 파일과 웹사이트가 동시에 팍! 뜸. (총 0.5초 소요)
    - **캡션:** _"HookLink (0.5초)"_

#### **카피라이팅 (Copywriting)**

이 영상 위에 얹을 문구입니다. "대중성"을 위해 어려운 말은 다 뺍니다.

- **H1 (헤드라인):**
    > "아직도 폴더를 더블클릭 하고 계신가요?" > 
    > (Are you still double-clicking folders?)

- **H2 (서브 헤드라인):**
    > **"마우스는 내려두세요. 당신이 찾던 파일과 웹사이트, 단축키 한 번이면 충분합니다."**

- **CTA 버튼:**
    > 지금 0.5초의 세계 경험하기 (무료 다운로드)
    > 밑에 작게: macOS 전용 | M1, M2, M3 최적화

### 2. 설득의 흐름 (Persuasion Flow)

히로 섹션 아래로 이어지는 스크롤 흐름입니다.

#### **Step 1. Pain Point (공감 유도)**
- **카피:** "우리는 인생의 19%를 '파일 찾기'에 낭비합니다."
- **내용:** 브라우저 북마크는 쌓여만 가고, 다운로드 폴더는 엉망입니다. 정리가 안 돼서가 아니라, **'접근 방식'이 잘못되었기 때문**임을 지적합니다.

#### **Step 2. The Solution (속도와 쾌감)**
- **핵심:** "키보드에서 손을 떼지 마세요."
- **기능 강조:**
    - **초성 검색:** 파일명이 기억 안 나도 `ㅂㄱㅅ`(보고서)만 치면 됩니다.
    - **통합 검색:** 웹사이트든 PDF든 차별하지 않습니다.
    - **태그:** 폴더 정리가 귀찮으면 그냥 태그(`#중요`) 하나만 달아두세요.

#### **Step 3. Comparison Table (비교 우위)**

말씀하신 대로 **훅마크 이름은 가리고(Blind)**, '구독형 전문가 도구'라는 카테고리로 묶어서 팹니다.

| **특징**  | **기존 방식 (Finder/Chrome)** | **구독형 연결 도구 (Competitors)** | **HookLink (우리 앱)**     |
| ------- | ------------------------- | --------------------------- | ----------------------- |
| **속도**  | 느림 (마우스 필수)               | 보통 (스크립트 로딩)                | **즉시 반응 (Rust 엔진)**     |
| **가격**  | 무료                        | **매년 $70 (구독)** 💸          | **커피 한 잔 값 ($4.99)** ☕️ |
| **데이터** | 파편화됨                      | **암호화된 DB (폐쇄적)**           | **투명한 텍스트 파일 (내 것)**    |
| **사용성** | 익숙함                       | 매우 복잡함 (전문가용)               | **직관적 (누구나 사용)**        |

> **전략 포인트:** 훅마크를 깎아내릴 때 "기능이 많아서 느리다"보다는 **"비싸고 내 데이터를 가둔다"** 는 점을 강조하여 대중의 '반감'을 자극합니다.

#### **Step 4. Data Sovereignty (데이터 주권 - 대중적 버전)**

"데이터 주권"이라는 말은 좀 어렵습니다. 대중에게는 **"안심(Safety)"**으로 접근합니다.

- **카피:** **"당신의 기록은 당신의 컴퓨터에만."**
- **설명:** "HookLink는 당신의 데이터를 클라우드로 가져가지 않습니다. 암호화해서 숨기지도 않습니다. 평범한 텍스트 파일로 저장되니 안심하세요. 앱을 지워도 기록은 남습니다."

#### **Step 5. Pricing (가격 정책 - 박리다매의 정점)**

마지막 결정타입니다.

- **Hook:** **"구독료 0원 선언."**
- **카드 섹션:**
    - ~~정가 $9.99~~
    - **얼리버드 특가 $4.99 (평생 소장)**
    - "매달 나가는 구독료가 아닙니다. 햄버거 하나 값으로 평생의 쾌적함을 사세요."

### 3. 추가 조언: "일반 대중"을 잡기 위한 한 끗

**1. 용어 순화 (Wording)**
- Deep Link (딥링크) → **바로 가기 주소**
- Bidirectional (양방향) → **(언급 삭제하거나) 연결된 파일**
- Graph/Mesh → **(언급 삭제)**
- Context → **생각의 흐름**

**2. 윈도우 사용자 차단 (필터링)**
- 이 앱은 macOS 전용입니다. 대중을 타겟팅하면 윈도우 사용자가 와서 "왜 안 돼요?"라고 묻습니다.
- 히로 섹션 버튼 옆에 **"Designed for macOS"** 를 아주 명확하게 박아주세요.

이 전략대로라면, 훅마크를 모르는 사람도 **"어? 내 맥북 쓰는 방식이 답답했던 거구나. 5천 원이면 속는 셈 치고 써볼까?"** 하고 지갑을 열게 될 것입니다.
